CarbonSavvy: Building Tomorrow's Green Cities

Description:
CarbonSavvy is a machine learning-based project dedicated to fostering sustainable urban development by predicting and managing CO2 emissions in cities. Leveraging Random Forest Regression, the system analyzes various parameters to forecast CO2 emission percentages. These parameters include resource availability for Wind Energy, Solar Energy, Nuclear Energy, Thermal Energy, and Bio Energy, alongside factors such as vehicle types, waste management, population density, forest coverage, and industrial presence. The project features an interactive GUI for easy access and quick predictions.

Keywords:
CO2 emissions, Green Cities, Economic Growth, Foresting, Sustainable Energy, Machine Learning, Industry Structure, Forecasting

Installation
1. Ensure Python is installed on your computer. Download Python from the [official website](https://www.python.org/downloads/).
2. Clone the project using the following command:  
   `git clone https://github.com/AmarNathPrajapati/CarbonSavvy_`
3. Install the required libraries using:  
   `pip install -r requirements.txt`
4. Open the `main.py` file.
5. Run the Python code via terminal using:  
   `python3 main.py`
6. The software will run on the default port 5000.
7. Click on the URL in the terminal with Ctrl pressed or type `http://localhost:5000` in your browser.

By providing a user-friendly platform, CarbonSavvy empowers individuals, city planners, and policymakers to make informed decisions towards creating environmentally sustainable cities. Through the integration of machine learning and an intuitive interface, CarbonSavvy represents a valuable tool for fostering environmental awareness and guiding urban development towards a greener and more sustainable future.
